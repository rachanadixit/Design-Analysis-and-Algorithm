# Design & Analysis of Algorithms Laboratory  
_B. Tech. Information Technology, Semester V_  
**Course Code:** BIT25PC04

## Overview

This repository contains assignments and code solutions for the **Design & Analysis of Algorithms Laboratory**. The course is designed to help students develop practical skills in analyzing, designing, and implementing efficient algorithms to solve real-world problems, with an emphasis on computational complexity, design paradigms, and optimization strategies.

---

## Course Objectives

1. Understand the basics of computational complexity for various algorithms.
2. Select and apply appropriate algorithm design strategies for real-world problems.
3. Comprehend and classify polynomial-time, NP, NP-complete, and NP-hard problems.

---

## Course Outcomes

On completing this laboratory, students will be able to:

1. Analyze the asymptotic performance of algorithms.
2. Solve computational problems using Divide & Conquer and Greedy techniques.
3. Apply Dynamic Programming for optimal solutions.
4. Use Backtracking and Branch & Bound strategies to solve complex problems.
5. Classify and distinguish P, NP, NP-complete, and NP-hard problems.

## Laboratory Assignments

Below is a list of suggested assignments reflecting real-world applications of classical algorithms.

| Sl. | Assignment | Algorithmic Focus |
|-----|------------|------------------|
| 1 | **Customer Order Sorting:** Implement Merge Sort for arranging customer orders with efficiency analysis vs. traditional methods. | Merge Sort, Complexity Analysis |
| 2 | **Movie Recommendation Optimization:** Use Quicksort to efficiently sort movie datasets on user criteria. | Quicksort, Large Data Handling |
| 3 | **Emergency Relief Supply Distribution:** Apply Fractional Knapsack to maximize utility of supplies with weight constraints. | Fractional Knapsack, Greedy |
| 4 | **Smart Traffic Management:** Implement Dijkstra’s Algorithm for real-time shortest path navigation for ambulances. | Dijkstra’s Algorithm, Graphs |
| 5 | **Logistics Delivery Optimization:** Model network as a multistage graph, use Dynamic Programming or Dijkstra to optimize routes. | Dynamic Programming, Graphs |
| 6 | **Disaster Relief Allocation:** Model resource allocation as a 0/1 Knapsack problem; compare Brute Force, DP, and Greedy solutions. | 0/1 Knapsack, Brute Force, DP, Greedy |
| 7 | **University Exam Scheduling:** Graph coloring for exam slot allocation, with comparison of coloring algorithms. | Greedy Coloring, DSATUR, Welsh-Powell |
| 8 | **Delivery Route Optimization (TSP):** Use Branch & Bound to efficiently solve the Traveling Salesman Problem for logistics. | Branch & Bound, TSP |

> _Assignments are designed for large datasets and real-world constraints including dynamic updates, prioritization, and efficiency optimization._
